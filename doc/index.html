Documentation Overview

Both the Lispel implementation and this documentation are in a very preliminary state right now. Refer to the language reference for a list of default commands available and simply try them in the standalone interpreter to check out the details. I will try to enhance the documentation as soon as the implementation is in a more stable state.

Table of Contents

  • Overview
    Explains in short what Lispel is and why you may want to give it a try.
  • Using the Interpreter
    Gives a few short examples of using the standalone interpreter.
  • Language Reference
    A list of the commands available by default. The commands available might differ from application to application.
  • Interpreter Design
    Explains what components make up the interpreter and how they interact.
  • Embedding Lispel
    Demonstrates how to integrate Lispel support in your C++ applications.
  • API Reference
    A reference manual for the various classes making up the interpreter. The API documentation is automatically generated from the sources using Doc++.
  • Glossary
    Descriptions of the terms used throughout the Lispel documentation.
